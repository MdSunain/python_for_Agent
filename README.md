# Movie Recommendation System

A content-based movie recommendation system built using the TMDB 5000 Movies dataset.
The system recommends similar movies by analyzing textual metadata such as overview, genres, keywords, cast, and crew.

The pipeline follows a structured ML workflow:

### Data → Preprocessing → Feature Engineering → Vectorization → Similarity Calculation → Recommendation

---
## Tech Stack
1. Python
2. NumPy
3. Pandas
4. Scikit-learn
5. NLTK

## Installation & Usage
1. Clone the repository
 `cd movie-recommender`
2. Run the script
`recommend("Batman Begins") `
